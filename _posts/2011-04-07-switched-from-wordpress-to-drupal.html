Today I migrated this site from Wordpress to Drupal. Nowadays I mostly do programming related to Drupal so it seemed like the right thing to do. The content migration went smooth using the Drupal modules Feeds and Feeds XPath Parser. Using these modules I was able to create nodes from the standard WordPress eXtended Rss export file.

There is still some tinkering left to do like find a more exciting theme and perhaps setup some kind of archive. Well, we'll see what happens.
